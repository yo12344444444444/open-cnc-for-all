# v1-shaft-bearing

The most salvage-friendly build. 8 mm smooth rods + LMK8UU bearings on every axis, driven by 8 mm × 2 mm-pitch lead screws.

## Why this build

- Almost every part can come from a dead i3-style 3D printer.
- Cheapest option if you're starting from nothing.
- Easy first build — if you've assembled an i3 printer, you can do this.

## Trade-offs

- Lower stiffness than rail-based builds. Best for wood, foam, soft plastic, light aluminum.

## What's in this folder

- `Assem1.SLDASM` — the full assembly, open this in SolidWorks
- `z/Z_cartige.SLDASM` — Z-axis sub-assembly
- All the `.SLDPRT` files — the printable parts and modeled hardware

## What you'll need to build it

Salvaged from old printers:
- 8 mm smooth rods (×6, lengths in the BOM)
- LMK8UU linear bearings (×8)
- T8 × 2 mm lead screws + brass nuts (×3)
- NEMA 17 steppers (×3)
- 5×8 flexible couplers (×3)
- Endstops, wiring, 12/24 V PSU, stepper drivers

To buy:
- HIWIN HG15 rail + HGH15CA carriage (used on one axis)
- Spindle / router (any cheap 500 W is fine)

## Build difficulty

⭐⭐ Beginner-friendly.
